# 🚀 ActiveFlow AI - The Last Minute Life Saver

> Your AI partner that saves you before deadlines do.

ActiveFlow AI is an AI-powered productivity companion designed to help students, professionals, and entrepreneurs plan, prioritize, and complete tasks before deadlines are missed.

Unlike traditional productivity tools that only send reminders, ActiveFlow AI actively analyzes user goals, creates smart action plans, predicts deadline risks, and provides recovery strategies when users fall behind.

Built using Google AI technologies, ActiveFlow AI works as an intelligent productivity assistant that transforms last-minute stress into structured execution.

# 🎯 Problem Statement

## The Last-Minute Life Saver

Students, professionals, and entrepreneurs frequently miss important commitments:

- Assignments
- Projects
- Meetings
- Payments
- Interviews
- Personal goals

Most productivity applications are passive. They only remind users about tasks but do not actively help them complete those tasks.

ActiveFlow AI solves this problem by becoming an active AI productivity partner that understands context, prioritizes tasks, and helps users take meaningful action.

---

# 💡 Solution Overview

ActiveFlow AI uses Gemini AI to convert natural language goals into structured execution plans.

Users can simply describe their situation, and ActiveFlow AI:

- Understands the goal
- Breaks goals into smaller actionable tasks
- Prioritizes important activities
- Estimates effort and time
- Creates personalized schedules
- Predicts deadline risks
- Suggests recovery strategies

The goal is to help users move from last-minute panic to controlled progress.

---

# ✨ Key Features

## 🧠 AI Task Analyzer

Users can enter goals in natural language.

Example:

User Input:
"I need to complete my machine learning project before Monday."

AI Output:

- Goal summary
- Task breakdown
- Priority level
- Estimated effort
- Recommended timeline

---

## 📊 Deadline Risk Prediction

ActiveFlow AI analyzes:

- Deadline
- Remaining work
- Task complexity
- Available time

and predicts whether the user is likely to miss a deadline.

Example:

Risk Level: HIGH

Reason:
Current workload is greater than available time.

Recommendation:
Focus on critical features first and reduce unnecessary work.

---

## 📅 AI Daily Planner

Creates a personalized daily productivity plan based on:

- Urgency
- Importance
- Deadlines
- User preferences

Example:

Today's ActiveFlow Plan:

09:00 - Complete coding module

13:00 - Prepare documentation

18:00 - Review progress

The AI explains why each task is prioritized.

---

## 🚨 Last Minute Rescue Mode

A special AI feature designed for emergency situations.

When users are close to a deadline, ActiveFlow AI creates a recovery strategy.

Example:

24 Hour Rescue Plan:

0-6 Hours:
Complete core functionality

6-12 Hours:
Testing and improvements

12-24 Hours:
Final review and submission preparation

---

## 🤖 AI Productivity Coach

An interactive AI assistant that helps users:

- Break down overwhelming tasks
- Identify blockers
- Create realistic plans
- Stay focused
- Make better productivity decisions

---

## 🧩 Productivity Memory

ActiveFlow AI learns user patterns and preferences.

Example:

User usually works better at night.

Recommendation:
Schedule difficult tasks after 8 PM.

This helps provide personalized suggestions over time.

# 🚀 Getting Started

Create a .env file:

VITE_GEMINI_API_KEY=your_api_key

## Run Development Server

npm run dev
